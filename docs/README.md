## 🧠 Tecnologias Principais
- **Frontend:** Next.js 14 (App Router), TailwindCSS, Framer Motion  
- **Backend:** NestJS, TypeORM  
- **Banco de Dados:** MySQL ou PostgreSQL  
- **Autenticação:** JWT  
- **Design:** Galileo AI
